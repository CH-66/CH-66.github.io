二、访问数据库

1、列举数据库：\l
2、选择数据库：\c 数据库名
3、查看该某个库中的所有表：\dt
4、切换数据库：\c interface
5、查看某个库中的某个表结构：\d 表名
6、查看某个库中某个表的记录：select * from apps limit 1;
7、显示字符集：\encoding
8、退出psgl：\q

======================================

9、列出当前数据库所有表
\dt

10、列出表名
SELECT tablename FROM pg_tables;
WHERE tablename NOT LIKE 'pg%'
AND tablename NOT LIKE 'sql_%'
ORDER BY tablename;

11、列出数据库名
\l或SELECT datname FROM pg_database;

===============================

三、postgresql数据管理系统使用命令方式有两种:

  1. 内部命令,以反斜线开始 \ ,如: \l 显示所有数据库
  2. 标准SQL命令,以分号 ; 或 \g 结束,可以使用多行
